🌿 Mental Health Awareness Website
🧠 Project Overview
This website is designed to offer beginner-friendly, accessible information about mental health in a calm, welcoming environment. It provides guidance on recognizing common mental health issues, managing stress, and encourages positive mental well-being through supportive design and content.

🎯 Goals
External User’s Goal
To access straightforward, supportive, and beginner-friendly mental health information. Users can expect:
* Clear explanations of common mental health challenges.
* Practical stress management tips.
* A calming, easy-to-navigate layout with uplifting messaging.
Site Owner’s Goal
To create a simple, welcoming webpage using HTML, CSS, and Bootstrap that delivers:
* Basic mental health information.
* A supportive design and structure.
* Visually appealing components using Bootstrap for consistency and accessibility.

🛠️ Technologies Used
* HTML5
* CSS3
* Bootstrap 5

🌟 Key Features
✅ Hero Section with Positive Messaging
* Uses a Bootstrap Jumbotron (or similar component) to welcome users.
* Features a calm color scheme and gentle background image.
* Includes an uplifting and encouraging message about mental health.
✅ Information Cards
* Leverages Bootstrap Cards to display:
    * Mental health tips
    * Brief overviews of common issues (like anxiety, stress, or depression)
* Provides digestible content in a clean, organized layout.
✅ Resource Links
* Displays external resource links in a Bootstrap grid layout.
* Styled with Bootstrap buttons to draw attention and encourage exploration.
✅ Positive Affirmations
* Features a section with uplifting quotes or affirmations.
* Utilizes Bootstrap text utilities for typography styling and spacing.
